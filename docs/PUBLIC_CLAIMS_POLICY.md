# 对外主张政策（公开墙 + 证据闭环）

## 公开墙

对外只陈述：`我只对外陈述 artifacts/ 证据工件覆盖到的结论。`

## 证据绑定（必须）

任何出现“更强/更稳/更省/更易集成”的表述，必须同时提供：

- `REPRODUCE.md`（复现步骤）
- `artifacts/env.json`
- `artifacts/results.json`
- `artifacts/report.md`
- `artifacts/manifest.json`（完整性校验）

## 禁止项

- 不讨论/不暗示任何“隐藏上限”或“未来倍率承诺”
- 不做未来路线承诺
- 不出现锁死/惩罚第三方/不可破解等表述
